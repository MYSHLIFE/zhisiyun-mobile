zhisiyun-mobile
===============

Mobile version of zhisiyun.com

Setup and Install Bower Components
----------------------------------

bower install

build jQuery Mobile
-------------------
change to jquery mobile path, then issue 'npm install', after npm installed, then 'grunt'.


Build
-----
build all
---------
gulp build

build css
---------
gulp css

build js
--------
gulp js

clean
-----
gulp clean
